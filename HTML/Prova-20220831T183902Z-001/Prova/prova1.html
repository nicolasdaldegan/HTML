Navegadores de internet

Os navegadores são provedores de acesso que foram criados para facilitar o uso da internet
sabemos que hoje a internet é um dos serviços mais usados da atualidade, des de um mega
computador até um simples celular ela está presente

Existem diversos tipos de navegadores da WEB e novos surgem todos os dias porém alguns se mantem
no topo, os mais usados da atualidade são

  • Internet Explorer
  • Google Chrome
  • Firefox
  • Opera
  • Safari
  • Android browser
